# Funkalphabet – SRC Lernkarten

A simple flashcard app to learn the radio/aviation alphabet (ICAO phonetic alphabet) and numbers for SRC (Short Range Certificate) exam preparation.

## Features

- **Letters A–Z** and **numbers 0–9** with their phonetic spellings (Alpha, Bravo, Charlie, …)
- Spaced repetition: cards you get wrong require 2 correct answers to master
- Progress tracking with a visual overview
- Keyboard shortcuts: Space to reveal, ←/F for wrong, →/J for correct
- Works offline – single HTML file with no build step

## Usage

Open `funkalphabet.html` in any modern browser. No server or installation required.
